Coffee Data Collection & Analysis

A comprehensive system for scraping coffee prices, transforming CSV datasets, syncing to MySQL, and exploring the Vietnamese coffee market via notebooks and scripts.

Beautiful Soup Selenium MySQL (Aiven) Jupyter EDA Vietnam Market
Project
ADY 201
Focus
Scrape → Process → Analyze
DB
MySQL

Features

  • Multiple scraping approaches
  • Automated MySQL sync
  • Processing pipelines
  • Fallback data generation
  • EDA with notebooks
  • Error handling & validation

Project Structure

coffee_dabase/
├─ Data Files (CSV)

├─ Analysis Notebooks (EDA)
├─ Testing (pytest style)
└─ Documentation (README, VN guide)

Data Highlights

  • Prices from giacaphe.com
  • Production, weather, export statistics
  • Market trade by importer country

Usage

Web Scraping

# Simple (Beautiful Soup)
python coffee_price_scraper.py

# Selenium (Cloudflare)
python coffee_price_scraper_selenium.py

# Complete with fallback
python coffee_price_scraper_final.py

Scrapers attempt to fetch latest prices, persist CSVs with timestamps, and generate fallback data when necessary.

Database Sync

python sync_coffee.py

Reads CSVs, transforms wide→long, creates/updates tables, and upserts into MySQL.

Analysis

jupyter notebook eda_data_coffee.ipynb
jupyter notebook main_coffee.ipynb

Setup

Prerequisites

  • Python 3.8+
  • MySQL / Aiven MySQL
  • Chrome + ChromeDriver

Install

pip install pandas pymysql sqlalchemy 
beautifulsoup4 selenium requests 
webdriver-manager python-dotenv

.env

HOST=your-host
PORT=3306
USER=your-user
PASSWORD=your-pass
DB=your-db
CA_PEM=/path/to/ca.pem

Scrapers

  • coffee_price_scraper_final.py – robust, fallback, timestamps
  • coffee_price_scraper_selenium.py – bypass Cloudflare
  • coffee_price_scraper.py – simple BS4

Processing

  • sync_coffee.py – wide→long, normalize, upsert
  • unprocessing_sql.py – raw processing

Testing & Docs

Data Sources

  • Coffee prices: scraped from giacaphe.com (with fallback)
  • Production, weather, export: Data_coffee.csv

Requirements

  • pandas, pymysql, sqlalchemy
  • beautifulsoup4, selenium, requests
  • webdriver-manager, python-dotenv
  • (Notebooks) jupyter, ipykernel

Configuration Notes

  • Update CSV paths in sync_coffee.py, main_coffee.ipynb, unprocessing_sql.py.
  • MySQL connection via .env; ensure utf8mb4 charset.
  • ChromeDriver is auto-managed; configure headless/timeouts if needed.
  • For Aiven MySQL, provide CA_PEM certificate path.
